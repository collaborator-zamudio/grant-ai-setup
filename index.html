Grant's AI Computer Setup Plan

A 7-part implementation roadmap covering hardware, agent setup, Notion, voice, email, contacts, and cloud architecture.

1. Hardware recommendation Buy now ▾
Recommendation: MacBook Air M5 (24GB RAM, 512GB SSD) — ~$1,299. Skip the MacBook Neo. It's an entry-level budget model for students, with only 8GB RAM — not suitable for running AI agents. The Air M5 is the sweet spot for cloud-connected AI workflows.
  • 1
    RAM is the #1 spec. 16GB is the minimum; 24GB is the future-proof choice for running local models and parallel agents. 8GB (Neo) will bottleneck immediately.
  • 2
    M5 chip advantage: Every GPU core has a dedicated Neural Accelerator — 4× faster AI compute than M4. Whisper voice transcription runs in under 400ms locally on M5 Apple Silicon.
  • 3
    Desktop alternative: Mac mini M4 Pro (48GB) at ~$1,799 is the best AI workstation per dollar — faster, more RAM, stays at a desk paired with any monitor.
  • 4
    Storage: 512GB minimum. Agents, local models, and filesystem access all grow fast. 1TB if budget allows.

The MacBook Neo (A18 Pro, $599) is Apple's budget laptop for students — 8GB RAM, no MagSafe, no Thunderbolt, and does not meet Adobe's RAM recommendations. It is not the right machine for this use case.

2. Agent setup — Claude & Gemini on Mac Setup steps ▾
  • 1
    Install Claude Desktop from claude.com/download. Sign in with a paid plan (Pro at $20/mo minimum). This gives access to Chat, Cowork, and Code modes in one app.
  • 2
    Enable Claude Cowork (Anthropic's non-technical desktop agent, launched Jan 2026). In the app, click the "Cowork" tab → check "Work in a Folder" → select a folder. Claude will autonomously plan and execute multi-step tasks on files in that folder.
  • 3
    Filesystem access via MCP: The Filesystem MCP Server (officially supported) lets Claude read, write, and organize local files. Configure it in Claude Desktop settings — you control exactly which folders are accessible. Over 9,400 MCP servers now exist covering email, databases, calendar, and more.
  • 4
    Email access: Connect Claude to Gmail or Outlook via MCP server. For personal email, the Gmail MCP server allows Claude to read, draft, and send on Grant's behalf. Grant must authorize OAuth access during setup.
  • 5
    Gemini: Use Google's Gemini app (gemini.google.com) or Gemini CLI for agent tasks. Gemini is strongest when deeply integrated with Google Workspace (Gmail, Drive, Docs).
Security note: Grant AI agents access to folders/email in stages, not all at once. Start with a single test folder. Limit permissions incrementally as trust is established.

3. Notion as AI agent platform Already using ▾
Notion is a strong hub for this setup. With the September 2025 launch of Notion 3.0 (built-in AI agent), the May 2026 Developer Platform (external agent integration, Workers, database sync), and Notion Mail, it's evolved into a capable personal operating system.
  • 1
    Notion AI agent (v3.0): Activate from any Notion page. The built-in agent can search across your workspace, summarize pages, create database entries, and run multi-step tasks — no additional setup required on a paid plan.
  • 2
    Connect Claude Code to Notion: As of May 2026, Notion's Developer Platform supports direct external agent integration — Claude Code can be assigned tasks and tracked inside Notion workspaces. This lets Grant kick off Claude tasks from within Notion and see results logged there.
  • 3
    Notion as CRM + contacts hub: Build a People database in Notion (name, email, company, relationship notes, last contacted). This becomes the enriched contacts layer fed from the iCloud export in step 6.
  • 4
    Notion Mail: Launched April 2025 as a Gmail-based email client with AI integration — draft replies, schedule meetings, and search across messages, all inside Notion.
  • 5
    Database Sync (May 2026): Notion can now pull live data from external sources directly into Notion databases. Useful later if Grant's contact enrichment or CRM data lives in other tools.
  • 6
    Superwhisper + Notion: Superwhisper works system-wide, so Grant can dictate notes, tasks, and meeting summaries directly into any Notion page via voice — no typing required.

4. Whisper Voice integration Setup steps ▾
Recommended app: Superwhisper. The leading Whisper-based voice input layer for Mac. Runs entirely on-device (no audio leaves the machine), works system-wide in any app including Claude Desktop and Notion, and has official integrations with Claude Code as of 2026.
  • 1
    What Whisper is: OpenAI's open-source speech recognition model. Apps like Superwhisper package it as a native Mac tool, converting voice to text in any app — email, Claude, Notion, Slack — in under 400ms on Apple Silicon.
  • 2
    Install Superwhisper (superwhisper.app). Pricing: $8.49/month or $249 lifetime. Set a hotkey (or mouse button) to activate dictation from any app.
  • 3
    Create custom modes for Grant's workflows: an "Email" mode that polishes spoken drafts, a "Claude prompt" mode that formats voice input as a precise instruction, and a "Notes" mode for quick capture into Notion.
  • 4
    Claude Code integration: Superwhisper has a native Claude Code mode (added 2026). Grant can dictate coding tasks or agent instructions directly into the Claude terminal.
  • 5
    Alternative — Wispr Flow: More of a "cloud polish" approach. Better for prose writing; Superwhisper is better for precision commands and technical workflows.

5. Work email separation (Exchange → web) Quick win ▾
  • 1
    Remove Microsoft Outlook (or Apple Mail's Exchange account) from the personal Mac. This frees the machine from work credentials entirely.
  • 2
    Access work Exchange via browser: Go to outlook.office.com. All Exchange email, calendar, and contacts are accessible without any desktop client. Works in Safari or Chrome.
  • 3
    Create a dedicated browser profile for work — keeps work cookies/sessions separate from personal browsing and prevents cross-contamination with personal AI agents.
  • 4
    Personal email client stays native: Use Apple Mail or Mimestream (Gmail-native Mac app) for personal email. This is the account Claude agents get access to — not work Exchange.
  • 5
    Optionally use Notion Mail for personal email — Gmail-based with built-in AI to draft replies, summarize threads, and schedule meetings, all inside Notion.

This is the fastest step in the whole plan. Takes under 30 minutes. Do this first — it cleanly separates the two environments before any agent setup begins.

6. iCloud contacts → Notion pipeline Setup steps ▾
  • 1
    Export from iCloud: Open Contacts.app on Mac → Select all (Cmd+A) → File → Export → Export vCard (.vcf file). This captures all contacts in a portable format.
  • 2
    Convert vCard to CSV: Use a free tool like vcf-to-csv.com or the macOS "Contacts" export to CSV option. Produces a spreadsheet with name, email, phone, and company fields.
  • 3
    Import to Notion: In Notion, create a new database → Import → CSV. Map columns to Notion properties (Name, Email, Phone, Company, Tags). This becomes Grant's personal CRM.
  • 4
    Deduplication and cleansing: Use SyncPenguin (syncpenguin.com) for ongoing two-way sync between iCloud Contacts and Notion. For a one-time cleanse, Claude Cowork can scan the Notion database and flag/merge duplicates.
  • 5
    Enrichment: Use Clay.com — connect it to the Notion database and it will auto-fill company info, LinkedIn URLs, job titles, and email addresses for contacts missing data.
  • 6
    Ongoing sync: SyncPenguin or Appy Pie Automate can keep iCloud and Notion in two-way sync — new contacts added on iPhone automatically appear in Notion, and vice versa.

7. Cloud vs. two-computer model Recommendation ▾
Recommendation: Single Mac, cloud-first. One MacBook Air M5 with cloud AI (Claude, Gemini via APIs) as the primary agent layer. No local model downloads needed for Grant's use cases — Cowork, email, and Notion workflows all run cloud-side.
  • 1
    Cloud-first model (recommended): Claude and Gemini run on Anthropic/Google's servers. The Mac is a lightweight interface and local compute layer. Fast setup, no large model downloads, and agents are always on the latest version.
  • 2
    Local model option (optional, later): If Grant wants offline AI or maximum privacy, Ollama + LM Studio let him run 7–14B parameter models locally on a 24GB M5 Mac. Not necessary at first — add later if needed.
  • 3
    Two-computer model — skip it: Adds complexity without clear benefit. The split between "documents computer" and "personal computer" creates sync headaches. Everything fits on one machine with proper folder structure.
  • 4
    Cloud storage: iCloud Drive handles filesystem backup. Use iCloud for Documents and Desktop. Claude agents get access to a designated local working folder that syncs to iCloud automatically.

The "desktop as AI compute layer, everything else in the cloud" model is exactly right. One Mac, cloud agents, cloud storage. Simple to maintain, easy to back up, and the Mac M5 has ample headroom if local AI is added later.
